Informations

This website is realized following the guide lines of the WaSP.

All the pages are generated by a Perl script which uses a template for the formatting written in XHTML.

Graphic and typographic aspects are managed through CSS.

There is also an RSS feed and you can subscribe it directly within each page of the site by clicking the RSS link in the top right corner, or by following this reference:
http://accademiaharmonia.interfree.it/rss/en/harmonia.rss.
This feed contains all the scheduled cultural and musical events.

For a better and safe experience please consider to use a web standards compliant browser like Firefox, Opera and Safari.
